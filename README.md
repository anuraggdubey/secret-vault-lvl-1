# Secret Vault — Midnight L1

A zero-knowledge commit/reveal demonstration built on the Midnight Network. This DApp allows a user to "commit" a secret value to the blockchain and later "reveal" it. The zero-knowledge proof generated by the Lace wallet ensures that the transaction is verified on-chain without the secret ever leaking.

## Tech Stack
- **Smart Contract:** Midnight Compact (`.compact`)
- **Frontend:** Next.js 14 (App Router), React 18
- **Styling:** TailwindCSS, Custom CSS Variables
- **Wallet Integration:** Lace v4 DApp Connector API (`window.midnight`)
- **Blockchain SDK:** `@midnight-ntwrk/midnight-js`

## Directory Structure
- `/contracts` — Contains the `secret_vault.compact` smart contract.
- `/web` — The Next.js frontend application.
  - `/app` — Next.js App Router (Pages, Layout, Providers).
  - `/components` — Reusable React components (UI library).
  - `/lib` — Context providers and Midnight SDK client wrappers.
  - `/managed` — (Generated) TypeScript bindings and ZKIR circuits compiled from the Compact contract.

## Getting Started

### 1. Compile the Contract
You must compile the smart contract first to generate the TypeScript bindings. You can use the official Midnight Docker container to compile this on any OS:
```bash
docker run --rm -v "${PWD}/../contracts:/contracts" -v "${PWD}/managed:/managed" ghcr.io/midnight-ntwrk/compactc:latest compile /contracts/secret_vault.compact -o /managed/secret_vault
```

### 2. Install Dependencies
```bash
cd web
npm install
```

### 3. Run the Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You will need the **Lace Wallet** browser extension configured for the Midnight Testnet (Preview) to interact with the Vault.
